# Egypt Macroeconomic Dashboard

[![Streamlit App](https://static.streamlit.io/badges/streamlit_badge_black_white.svg)](https://share.streamlit.io/yourusername/egypt-macroeconomic-dashboard/dashboard.py)

## Overview
An interactive dashboard for monitoring and analyzing Egypt's key macroeconomic indicators from 2021-2025, with a focus on:
- Inflation (Consumer Price Index)
- Exchange rates (Egyptian Pound/US Dollar)
- Monetary policy (Central Bank interest rates)

This dashboard provides real-time visualizations, economic insights, and automated reporting to help understand Egypt's economic trajectory following significant monetary reforms.

## Features
- Interactive visualizations with Plotly
- Automatically generated economic reports with contextual analysis
- Correlation analysis between key economic indicators
- Historical trend analysis with quarterly granularity
- Responsive design for all device sizes

## Data Sources
The dashboard utilizes official data from the International Monetary Fund (IMF) Statistical Databases:
- Consumer Price Index (CPI) data
- Exchange Rates (ER) data
- Monetary and Financial Statistics on Interest Rates (MFS_IR)
- Period: 2021-Q1 to 2025-Q2 (projected)
